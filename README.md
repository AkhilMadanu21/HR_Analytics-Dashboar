# 📊 HR Analytics Power BI Dashboard

This project presents a comprehensive HR Analytics dashboard built using Power BI. It aims to provide key insights into employee data such as attrition rates, department-level analysis, salary distributions, gender diversity, tenure, and performance metrics.


## 📌 Key Metrics Overview

| Metric              | Value     |
|---------------------|-----------|
| Total Employees     | 300       |
| Avg Tenure (Years)  | 5.04      |
| Attrition Rate      | 23.3%     |
| Avg Salary          | 89.46K    |
| Gender Distribution | 54.7% Male / 45.3% Female |

## 🛠️ Tools Used

- Microsoft Power BI Desktop  
- DAX and Power Query  
- PDF exported for online preview  

---

## 🧠 Insights Derived

- Sales, HR, and Marketing have the highest attrition.
- Engineering and Finance show lower attrition but slightly lower average salaries.
- Most employees are rated between 3 and 4 in performance.
- Strong correlation observed between higher tenure and better performance.
